# 用户增长策略 (Growth Strategy)

## 1. 增长模型 (AARRR)

### Acquisition (获取)
**策略**: "工具引流，内容留存"
- **SEO 矩阵**: 针对 "AutoML", "Data Cleaning Tool", "Shapley Value Visualization" 等关键词撰写高质量技术博客。
- **场景化 Landing Page**: 针对 "销售预测"、"设备维护" 制作独立的落地页，精准捕获垂直流量。
- **免费小工具**: 剥离"数据体检报告"功能作为免费引流工具，用户上传 CSV 即可获赠一份质量报告。

### Activation (激活)
**策略**: "Aha Moment 前置"
- **预置数据集**: 注册后默认提供"泰坦尼克号"、"波士顿房价"等经典数据集，用户无需上传即可体验完整流程。
- **新手引导**: 3步引导 (上传 -> 清洗 -> 建模)，确保用户在 2 分钟内看到第一个预测结果。
- **样例报告**: 首次进入直接展示一份完美的分析报告，展示"最终能得到什么"。

### Retention (留存)
**策略**: "价值周期化"
- **周报订阅**: 每周自动运行模型，推送"本周预测日报"邮件，提醒用户回来查看。
- **数据监控**: 当数据分布发生漂移 (Drift) 时，触发报警通知。
- **学习中心**: 内置"数据科学小课堂"，教用户如何解读 Shapley 值，提升用户粘性。

### Referral (推荐)
**策略**: "互利共赢"
- **双向奖励**: 邀请同事注册，双方各获得 1 个月的"高级模型"使用权。
- **报告水印**: 免费版导出的报告带有 "Powered by LimiX" 水印和二维码。
- **协作共享**: 允许将分析结果生成公开链接 (View-only)，便于在社交媒体传播。

### Revenue (收入)
**策略**: "按价值分层"
- **Freemium**: 基础功能免费 (限制数据量 50MB, 仅限通用模型)。
- **Pro ($29/mo)**: 无限数据量，高级因果解释，导出 PDF。
- **Enterprise**: 私有化部署，定制清洗规则，专属客服。

## 2. 病毒式传播机制设计

### 机制一：公开分享的分析报告
- **触发点**: 用户完成一次成功的分析，发现有趣的结论。
- **载体**: 精美的 H5 报告或长图。
- **传播动力**: 用户为了展示自己的专业度 (Social Currency)。
- **闭环**: 报告底部有"我也要分析数据"按钮 -> 跳转 Landing Page。

### 机制二：数据挑战赛
- **形式**: 举办月度数据分析挑战赛 (如"预测下月比特币价格")。
- **激励**: 奖金 + 证书。
- **传播**: 参赛者为了拉票或展示成果主动传播。
