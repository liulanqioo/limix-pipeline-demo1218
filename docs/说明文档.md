# 项目说明文档

## 一、项目规划

### 1.1 项目背景
本项目旨在快速构建一个名为 "LimiX" 的数据科学平台演示原型（MVP）。核心目标是通过高保真的前端交互和可视化，向潜在客户或利益相关者展示产品的核心价值主张：从数据导入、质量评估、清洗、任务管理到结果解释和对比的全流程能力。

### 1.2 核心目标
- **快速验证**：通过 Mock 数据和前端原型，快速验证产品概念。
- **视觉冲击**：使用高质量的 UI 组件（Arco Design / Tailwind CSS）和图表（Recharts）展示专业性。
- **完整流程**：演示从数据到决策的完整链路。

### 1.3 技术栈
- **前端框架**：React + Vite
- **UI 组件库**：Arco Design + Shadcn UI (Radix UI + Tailwind CSS)
- **样式**：Tailwind CSS
- **图表**：Recharts
- **图标**：Lucide React

## 二、实施方案

### 2.1 目录结构
- `src/LimiPreview.jsx`: 核心演示页面，包含所有功能面板的逻辑。
- `src/WelcomePage.jsx`: 欢迎/登录页面，展示品牌形象。
- `src/mock/data.js`: 统一管理 Mock 数据，确保演示的一致性。
- `src/components/`: 可复用的 UI 组件。

### 2.2 关键功能模块
1.  **数据集导入 (Datasets)**: 模拟文件上传和版本管理。
2.  **数据质量评估 (Health)**: 展示数据分布、缺失、漂移等体检报告。
3.  **清洗方案 (Clean)**: 演示自动化清洗建议和操作。
4.  **任务管理 (Tasks)**: 模拟模型训练任务的创建和状态追踪。
5.  **结果分析 (Results)**: 展示模型性能指标（AUC, Precision-Recall 等）。
6.  **可解释性 (Explain)**:通过热力图和条形图展示特征重要性（Shapley values）。
7.  **交付 (Deliver)**: (已隐藏) 演示 API 生成、报告导出等交付能力。
8.  **对比 (Compare)**: 展示 LimiX 与其他竞品（如 AutoGluon, LLM）的优势对比。
