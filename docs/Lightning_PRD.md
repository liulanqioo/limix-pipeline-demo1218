# 闪电式产品需求文档 (Lightning PRD)

**版本**: v1.0  
**日期**: 2025-12-18  
**状态**: MVP 已开发 (Dev Complete)

## 1. 产品概述

### 1.1 产品愿景
打造一款**"从结构化数据到可解释决策"**的端到端流水线产品 (LimiX)，解决传统 AI 落地中"黑盒不可信、交付周期长、运维难"的痛点，帮助政务与企业客户快速实现数据价值闭环。

### 1.2 价值主张
- **快**: 从 CSV 到 API 服务化仅需分钟级配置。
- **透**: 内置 SHAP 归因与反事实模拟，让决策可解释。
- **稳**: 集成数据体检与漂移监控，保障长期运行稳定性。

### 1.3 目标用户
- **指挥中心调度员 (ToG)**: 需要依据数据做出快速、无责的派单决策。
- **产线工程师 (ToB)**: 需要快速定位良率下降原因并调整参数。
- **数据科学家**: 需要快速验证想法并交付可复用的基线模型。

## 2. 用户研究 (基于三问题调研法)

| 调研问题 | 核心洞察 | 功能映射 |
| :--- | :--- | :--- |
| **Why (目标)** | "我想不用写代码就能把 Excel 里的数据变成能用的预测模型。" | 特征工厂、自动任务训练 |
| **How (痛点)** | "最烦模型上线后不知道为什么这就判错了，领导问起来没法解释。" | ExplainPanel (归因/反事实)、ComparePanel (同口径对比) |
| **What (下一步)** | "有了模型我想直接生成报告发给领导签字，或者生成 API 给开发接。" | DeliverPanel (报告/API/订阅) |

## 3. 功能需求 (MVP)

### P0: 核心闭环 (已完成)
- **数据体检 (HealthPanel)**: 自动识别缺失、异常、漂移，生成可视化热力图。
- **任务训练 (TasksPanel)**: 支持分类/回归/填补任务，一键训练并对比指标。
- **结果展示 (ResultsPanel)**: 阈值-收益曲线，支持业务导向的阈值选择。
- **可解释性 (ExplainPanel)**: 全局特征重要性、单案 SHAP 归因、反事实模拟器。
- **交付服务 (DeliverPanel)**: 自动生成业务报告 (PDF Mock)、API 调用代码生成。

### P1: 体验增强 (规划中)
- **多场景适配**: 支持更多预设场景模板 (如金融风控、医疗诊断)。
- **高级清洗**: 交互式数据清洗规则配置。
- **协同标注**: 多人协作打标与反馈循环。

## 4. 技术考量 (基于闪电式开发)

### 4.1 技术栈
- **前端**: React + Vite + Arco Design + Tailwind CSS (保证开发速度与组件质量)
- **可视化**: Recharts (轻量、响应式)
- **Mock**: 本地 Mock 数据驱动 (优先验证交互，后接入后端)

### 4.2 性能约束
- 首屏加载 < 1.5s
- 交互响应 < 100ms (Mock 模式下)
- 支持 20万行数据的前端虚拟滚动预览

## 5. 验证与指标

### 5.1 验证计划 (Week 1)
- **目标**: 验证"可解释性"是否能打动非技术决策者。
- **方法**: 使用 Demo 进行 5 场客户演示，观察其对 `ExplainPanel` 和 `DeliverPanel` 的反应。
- **成功标准**: 至少 3 位客户主动询问"能否试用"或"怎么收费"。

### 5.2 核心指标 (North Star)
- **Time-to-Value (TtV)**: 用户导入数据到生成第一份可用报告的时间 (目标 < 15分钟)。
