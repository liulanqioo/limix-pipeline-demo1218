# 闪电式路线图 (Lightning Roadmap)

## 阶段一：MVP 快速验证 (Weeks 3-6) - 已完成
**目标**: 核心流程跑通，验证"清洗+建模+解释"闭环价值。

### 核心交付物 (Features)
- [x] **数据资产**: 支持 CSV 拖拽上传与预览。
- [x] **数据体检**: 基础的缺失值、分布可视化。
- [x] **自动化清洗**: 基于规则的简单清洗建议。
- [x] **自动化建模**: 集成 LimiX (Mock) 和 AutoGluon (Mock) 两种模式。
- [x] **因果解释**: 全局特征重要性热力图 (Shapley)。
- [x] **交互演示**: 高保真前端原型，模拟真实计算延迟。

### 关键指标
- 演示流程完成率 > 90% (无明显 Bug 阻断)。
- 用户对"因果解释"页面的停留时间 > 30s。

---

## 阶段二：V1.0 体验优化 (Weeks 7-10) - 进行中
**目标**: 提升可用性与信任度，根据种子用户反馈微调。

### 核心交付物 (Features)
- [ ] **增强解释性**: 增加单样本归因分析 (Local Interpretability)，支持"为什么这个客户会流失"。
- [ ] **交互式清洗**: 允许用户手动调整清洗规则，不仅仅是接受建议。
- [ ] **多场景适配**: 增加"空气质量预测"、"设备维护"等行业模板。
- [ ] **导出报告**: 生成 PDF/HTML 格式的完整分析报告 (Validation -> Result -> Explain)。
- [ ] **稳定性**: 优化前端大数据量下的渲染性能 (虚拟滚动)。

### 关键指标
- 核心功能 (清洗、建模) 的操作步数减少 20%。
- 导出报告的使用率 > 40%。

---

## 阶段三：V2.0 增长与规模化 (Weeks 11-16)
**目标**: 构建增长引擎，支持更多样化的数据和协作。

### 核心交付物 (Features)
- [ ] **增长引擎**: "邀请同事协作"功能，共享数据集和分析结果。
- [ ] **高级因果**: What-if 模拟器 (反事实推理)。
- [ ] **数据连接器**: 支持 MySQL/PostgreSQL 直接连接。
- [ ] **社区版**: 开放部分清洗规则库，建立用户社区。
- [ ] **API 交付**: 将模型一键部署为 API 服务。

### 关键指标
- 病毒系数 (K-factor) > 0.5。
- 月活跃用户 (MAU) 增长率 > 30%。
